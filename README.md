# Minecraft 论坛前端项目

基于 Vue3 + Vite + TypeScript 的前端项目

## 技术栈

- Vue 3
- Vite
- TypeScript
- Pinia（状态管理）
- Vue Router
- Element Plus（UI 组件库）
- Axios（HTTP 请求）
- @kangc/v-md-editor（Markdown 编辑器）

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 预览

```bash
npm run preview
```

## 项目结构

```
src/
├── api/           # API 接口封装
├── components/    # 组件
├── layouts/       # 布局组件
├── router/        # 路由配置
├── stores/        # Pinia 状态管理
├── views/         # 页面组件
├── App.vue        # 根组件
└── main.ts        # 入口文件
```
